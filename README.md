# 🌟 Pattern Generator and Number Analyzer

A simple Python console application that allows users to:

✅ Generate star (`*`) patterns  
✅ Analyze a range of numbers  
✅ Identify Odd and Even numbers  
✅ Calculate the sum of numbers in a range  
✅ Exit the program using a menu-driven interface  

---

# 📌 Features

## ⭐ Generate Pattern
Creates a right-angled triangle pattern using `*` symbols.

Example:
```text
*
* *
* * *
* * * *
```
---

## 🔢 Analyze a Range of Numbers
For a user-defined range:

- 🟢 Detects Even Numbers
- 🔵 Detects Odd Numbers
- ➕ Calculates the total sum

Example:
```text
Number 1 is odd
Number 2 is even
Number 3 is odd
Number 4 is even

Sum of all number from 1 to 5 is: 15
```

---

## 🚪 Exit Program
Safely exits the application.
